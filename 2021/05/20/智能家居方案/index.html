智能家居方案

智能家居 方案
智能家居
发布日期:   2021-05-20

关于HomeKit与米家智能家居部署方案设想

一、核心中心控制

核心中控为Homekit

优点

  1. Siri识别率更高
  2. IPad可做为中控面板
  3. 基于home assistant服务可以集成更多设备和自动化部署
  4. 语音控制 HomePod（mini）功能更加强大

缺点

  1. 支持HomeKit的配件相对较少
  2. home assistant有一定的技术难度，维护相对麻烦
  3. 产品较贵

方案

  1. 采用核心为HomeKit，辅助位米家的解决方案
  2. 设备采用米家和aqara的智能产品
  3. 尽可能的使用原生支持双平台的硬件

二、需要环境

  • 理想的完美部署环境，基于一个良好的弱电与强电的环境，合理的弱电与强电的布局。
  • 需要众多硬件的支持
  1. 9U或者12U机柜 x1台
  2. 飞塔防火墙x1台（这里以fortinet 200b举例）
  3. Cisco 3960交换机x1台
  4. 摄像头若干
  5. 硬盘录像机x1（取决于4）
  6. 服务器x1台
  7. 群晖NAS（非必选，可用服务器虚拟黑群晖）
  8. aqara人体传感传感器若干
  9. aqara门窗传感器若干
  10. aqara温度传感器x1
  11. aqara浸水传感器x2 （卫生间和厨房使用）
  12. aqara窗帘控制器
  13. aqara LED灯泡若干
  14. 小爱同学
  15. 米家智能扫地机器人pro
  16. 米家智能空气净化器
  17. 米家空气加湿器
  18. 智能晾衣机
  19. 米家洗衣机
  20. 米家纯净式智能加湿器
  21. 米家直流风扇
  22. 米家空调或者空调伴侣3（原生支持homekit）
  23. Wi-Fi6路由器（ap）若干或者支持mesh组网技术的路由器
  24. 索尼电视x1
  25. ps5x1
  26. xbox series x
  27. aqara智能门锁
  28. 其他（灯带 氛围灯灯）

准备工作

智能家居的一切基于网络，所以打造一个良好的网络环境尤为重要

做好IP地址规划 ，将服务器，防火墙，交换机，硬盘录像机等设备装入机柜，做好标签，使用理线架，配线架，进行理线。

  1. 根据Wi-Fi（ap）的情况设置防火墙，首先配置trust和untrust，定义一个wan接口，用于连接光猫，DHCP或者PPPOE。
  2. 定义lan，与虚拟交换机，根据自身情况划分lan接口与配置（DHCP）
  3. 做好路由表
  4. 将Cisco 3960最后一个port定义为trunk接入防火墙，根据需求划分VLAN，trunk，access， DHCP（如果防火墙未开启DHCP）开启POE。
  5. 如果有无线控制器（WLC）做好配置，接入交换机
  6. 将ap做好配置（开启2.4GHz）（console）接入交换机。
  7. 将各个房间的弱电网线接入交换机。
  8. 如果摄像头非无线的，同样接入交换机。

开始部署

部署服务器

ESXI部署

  1. 首先在vmware 官网上下载ESXI 镜像，完成后使用软碟通写入U盘
  2. 接入服务器，从U盘启动，进入安装进程。
  3. 安装完成后，配置用户名和密码，IP地址等
  4. 配置完成后会重启，完成后应用生效。

系统安装

  1. 在home assistant官网下载虚拟机OVA
  2. 将 下载好的ova添加到虚拟机。
  3. 创建虚拟机。
  4. 设置HA随ESXI启动自启动

部署NAS

  1. 根据局域网情况设置NAS
  2. 注册群晖账户，并登陆
  3. 创建磁盘阵列，配置存储
  4. 创建共享文件夹

部署智能硬件

部署智能音箱

  1. 将homepod部署到对应的位置，并且和apple id 配对，添加到家庭app中
  2. 将小爱同学部署到对应的位置，并且和小米账户配对，添加到米家app中

部署中控面板

  • 将IPAD进行改造，放置于门口的位置（磁吸方案，粘贴方案等）
  • 登录apple id

部署智能配件

  1. 将上述智能配件置放于对应位置，并且连接网络（部分设备只支持2.4ghz）
  2. 将智能配件分别添加到米家和家庭app中
  3. 部分硬件不是原生支持HOMEKIT

配置Home assistant

  1. 首次登陆HA WEB 页面会有一些设备可以自动集成
  2. HA 会生成一个HomeKit的二维码，使用家庭app扫码添加。
  3. 获取各个不支持HomeKit的智能硬件的IP地址和token
  4. 添加对应插件，修改conf，填入对应的IP地址与token
  5. 将PS5和Xbox按照插件说明添加到HA
  6. 将索尼电视按照插件说明添加到HA
  7. 配置完成后，重启HA服务。
  8. 部署完成，查看手机 家庭app中由HA桥接的智能配件。

场景和自动化设置

根据自己已有的设备和需求，设置自动化

Siri以及自动化

Siri的使用

  1. 可以通过语音控制使用任何支持Siri的apple设备来控制任何一个智能硬件
  2. 可以通过Siri执行自动化和场景

自动化与场景

  1. 在家庭app中，自动化，添加自动化配置。
  2. 例如：每天早上6:00 ，自动执行窗帘控制器，打开窗帘，HomePod开始播放起床音乐。
  3. 在家庭app中配置场景
  4. 例如，游戏模式。配置为，游戏模式，启动窗帘控制器关上窗帘，启动PS5或者XBOX，开启氛围灯，启动索尼电视并切换到HDMI信号源，如果扫地机器人在工作，执行回充。

场景与自动化补充 可以使用米家进行扩展

IPAD中控

位于门口，在出门离家 ，和回家进门时方便进行操作，例如，离家布防，关闭空调，风扇，加湿器，空气净化器等，扫地机器人开始扫拖。回家解除布防，开起净化器等，扫地机器人回充。
